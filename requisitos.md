# Requisitos - Helix Health

## Visao Geral

Sistema hospitalar completo para gerenciamento de pacientes, profissionais, atendimentos, prontuarios, exames, internacoes, estoque/farmacia, financeiro e pronto socorro.

## Usuarios e Acessos

- Login para medicos.
- Login para almoxarifado.
- Login para recepcao.
- Login para administrador.
- Login para enfermeiro.
- Login para paciente.
- Login para financeiro.
- Login para farmacia.
